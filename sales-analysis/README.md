# 📊 Sales Performance Analysis

## 🎯 Objective
Analyze sales trends and identify top-performing regions.

## 🛠 Tools Used
- Power BI
- Excel

## 📈 Key Insights
- Revenue increased by 18% in Q4
- Electronics category generated highest sales
- West region is underperforming

## 📌 Conclusion
Business should focus on improving marketing strategies in the West region.
