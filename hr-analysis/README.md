# 📊 HR Workforce Analysis

## 🎯 Objective
Analyze employee attrition and workforce trends.

## 🛠 Tools Used
- Power BI

## 📈 Key Insights
- Attrition rate: 22%
- Highest attrition in IT department
- Employees with <2 years experience leave most

## 📌 Conclusion
Company should improve retention strategies for new employees.
